# personal-ai-assistant
A Python-based personal AI assistant powered by the Groq API and Meta’s LLaMA model, capable of maintaining conversational interactions with users in real time.


## Project Overview
This project is a **Personal AI Assistant built using Python, the Groq API, and Meta's LLaMA model**.  
The assistant can interact with users and generate intelligent conversational responses in real time.

The goal of this project is to demonstrate how **Large Language Models (LLMs)** can be integrated into applications to build conversational AI systems.

---

## Features
- Conversational AI assistant
- Real-time response generation
- Maintains conversation flow
- Uses Groq API for fast inference
- Powered by Meta's LLaMA model
- Built using Python

## Installation

Clone the repository:

```bash
git clone https://github.com/wyashverma/personal-ai-assistant.git
```

Move to the project directory:

```bash
cd personal-ai-assistant
```

Install required dependencies:

```bash
pip install -r requirements.txt
```

---

## How to Run the Project

Run the Python script:

```bash
python ai_assistant.py
```

Then start chatting with the AI assistant in the terminal.

---

## API Key Setup

This project requires a **Groq API Key**.

Add your API key as an environment variable:

```python
import os
client  = Groq(api_key="Your_API_key")
```
